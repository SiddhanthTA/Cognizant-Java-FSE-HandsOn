# Financial Forecasting

## Objective
Predict future financial values using a recursive algorithm.

## Classes

- FinancialForecast.java
- ForecastTest.java

## Concepts Used

- Recursion
- Base Case
- Recursive Case

## Time Complexity

O(n)

## Space Complexity

O(n)

## Optimization

The recursive solution can be optimized by using an iterative approach, which reduces the space complexity to O(1). For recursive problems with overlapping subproblems, memoization or dynamic programming can also be used to avoid repeated computation.
